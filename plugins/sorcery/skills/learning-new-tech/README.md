# learning-new-tech

A coaching workflow for learning a programming language, framework, or platform by doing. Scaffolds a `learning/` directory with a flexible outline, a cross-session notes file, and one numbered lesson at a time — each lesson self-contained, with its own `start.sh` and `score.sh`.

Fires on phrasings like "I want to learn Rust", "build a curriculum for SwiftUI", "set up a learning plan for Postgres", and on follow-ups like "lesson 03 done" or "what's next?"

See [`SKILL.md`](SKILL.md) for the trigger description and the full runbook.
