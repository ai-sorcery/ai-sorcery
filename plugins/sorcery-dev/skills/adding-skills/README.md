# adding-skills

Checklist for authoring new skills in the sorcery plugin: picking the shape (new skill vs. extension vs. catalog entry), file layout, installer decisions, SKILL.md conventions, testing, and the review + commit flow. Fires when a contributor asks to "add a skill for X" or "help me author a sorcery skill."

See [`SKILL.md`](SKILL.md) for the trigger description and the full checklist.
